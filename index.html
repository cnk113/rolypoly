rolypoly.

This is the site for rolypoly, a method for identifyin trait-relevant gene annotations using GWAS summary statistics. A bit about what we do.

Installation

Installation is easy. First install the required R packages then install from the tarball. In the future rolypoly will live on Bioconductor or CRAN to make this step even easier.

Usage

Check out this vignette for basic usage (add link). Check out our preprint for details of the methods included in rolypoly.
